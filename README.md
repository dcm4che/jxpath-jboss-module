jxpath-jboss-module
===================

JBoss module for jxpath

Build
=====

* `mvn install`

Install
=======

Unzip the content of _target/jxpath-jboss-module-1.0.0-SNAPSHOT.zip_ into the jboss root directory, e.g.
```
unzip target/jxpath-jboss-module-1.0.0-SNAPSHOT.zip -d /jboss-as-7.1.1.Final/
```
